# Financial and Dynamic Econometrics – Project Portfolio

This repository contains a collection of laboratory projects, simulations, and analytical reports focused on time series analysis, volatility forecasting, and advanced econometric modeling. 

## Project Overview

| Project | Description | Preview Links |
| :--- | :--- | :--- |
| **1. Financial Data Properties** | Statistical analysis of financial time series data (Open, High, Low, Close, Volume). Implements and compares various statistical tests for distribution normality, including Shapiro-Wilk, D'Agostino-Pearson, and Kolmogorov-Smirnov tests. |[View PDF 1](https://github.com/Laszczyk/financial-and-dynamic-econometry-/blob/main/projket1/Sprawozdanie_1B_grupa_4.pdf)|
| **2. MA(1) Model Simulation** | Monte Carlo simulation of the Moving Average MA(1) process. Focuses on parameter estimation, analyzing empirical distributions via histograms/boxplots, and testing asymptotic normality. | [View PDF 2A](https://github.com/Laszczyk/financial-and-dynamic-econometry-/blob/main/projekt2/Sprawozdanie_2A_grupa_4.pdf)<br>[View PDF 2B](https://github.com/Laszczyk/financial-and-dynamic-econometry-/blob/main/projekt2/Sprawozdanie_2B_grupa_4.pdf)|
**3. ARMA(1,2) Estimation & Forecasting** | Parameter estimation using Conditional Maximum Likelihood (ML) via Excel Solver. Includes analytical derivation of process properties (Expected Value, Variance, ACF up to lag 10) and multi-step ahead point and variance forecasting. | [View PDF 3A](https://github.com/Laszczyk/financial-and-dynamic-econometry-/blob/main/projekt3/Sprawozdanie_3A_grupa_4.pdf)<br>[View PDF 3B](https://github.com/Laszczyk/financial-and-dynamic-econometry-/blob/main/projekt3/Sprawozdanie_3B_grupa_4.pdf)|
| **4. Volatility Modeling (GARCH)** | Comprehensive volatility forecasting using the ARMA-GARCH framework.<br>• Evaluates GARCH, EGARCH, and GJR-GARCH specifications.<br>• Compares Normal, Student's t, and Skewed t error distributions.<br>• Features robust rolling-window analysis (100, 200, 500 obs). | [View PDF 4A](https://github.com/Laszczyk/financial-and-dynamic-econometry-/blob/main/projekt4A/Sprawozdanie_4A_grupa_4.pdf)<br>[View PDF 4B](https://github.com/Laszczyk/financial-and-dynamic-econometry-/blob/main/projekt4B/Sprawozdanie_4B_grupa_4_NST.pdf)|
| **5. VAR(1) Models & HC Estimators** | Simulation-based study of Vector Autoregression VAR(1) models evaluating the performance of Heteroskedasticity-Consistent (HC) standard errors. Analyzes the empirical size and power of Granger causality tests under classical OLS vs. HC estimators. | [View PDF 5A](https://github.com/Laszczyk/financial-and-dynamic-econometry-/blob/main/projekt5/Sprawozdanie_5A_grupa_4_NST.pdf)<br>[View PDF 5B](https://github.com/Laszczyk/financial-and-dynamic-econometry-/blob/main/projekt5/Sprawozdanie_5B_grupa_4_NST.pdf)|
| **6. European Stock Markets (COVID-19)** | Empirical study analyzing shock transmission across six major European stock exchanges (DAX, AEX, BEL20, OMXS, OMXV, WIG20).<br>• Utilizes VAR models and Granger causality tests.<br>• Employs Chow and LR tests to confirm structural breaks caused by the 2020 pandemic. | [View PDF 6](https://github.com/Laszczyk/financial-and-dynamic-econometry-/blob/main/projekt6/Sprawozdanie_6AB_grupa_4_NST_komentarze%20od%20Wojtowicza%20(1).pdf) |
